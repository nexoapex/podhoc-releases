# What's New in Podhoc

This release includes new features, improvements, and fixes.

## New Features
- ci:  use SSM parameters for CloudFront cache invalidation on deploy (#267)
- voice:  custom voice upload, processing, and management (#228)
- stt:  SageMaker speech-to-text transcription fallback (#258)
- smartwatch apps, E2E tests, podcasts-sync API, email redesign (#102)
- storybook:  visual component testing across all webapps (#257)
- vibevoice:  self-hosted TTS via SageMaker async inference (#141)
- GA4 event tracking for presentational-web and user-webapp (#253)
- Add missing Lambda + API Gateway definitions for playback-position, podcasts-recent, podcasts-shared-with-me (#254)
- smartwatch sync API and native apps for 4 platforms (#180)
- CarPlay and Android Auto app implementation (#230)
- Multi-currency pricing with EUR fallback (#244)
- Spotify metadata + lyrics extraction (LRCLIB + Lyrics.ovh) (#241)
- Multi-language 1-minute podcast samples (8 languages) (#243)
- recreate podcast in a different language (#227)
- automate browser extension CI/CD with store submissions (#178)
