# What's New in Podhoc

This release includes new features, improvements, and fixes.

## New Features
- async FLAC-to-MP3 transcode + Latin-language duration fix
- podcast deep-link playback with highlighting
- add browser extensions section with i18n support
- Firefox extension promotion in Wishlist views
- per-language OG images, optimized og:title, og:logo tag

## Bug Fixes & Improvements
- hide mobile menu in critical CSS to prevent FOUC
- resolve remaining PageSpeed issues for presentational web
- PageSpeed improvements — CLS, reflows, contrast, minification
