# What's New in Podhoc                                             
                                                                                                                                        
  This release introduces the Podhoc browser extension for Chrome and Firefox, a dedicated support page, and several improvements to the
   website experience.                                                                                                                  
                                                                                                                                        
  ## New Features                                                    

  - **Browser extension for Chrome and Firefox** — Save any webpage to your Podhoc wishlist directly from your browser with one click.  
  Just click the Podhoc icon, preview the page, and hit "Add to my wishlist."
  - **Extension prompt in the app** — A new banner on your Wishlist page helps you discover and install the browser extension for your  
  browser.                                                                                                                              
  - **Support page** — A dedicated support page is now available at podhoc.com/support, fully translated in 26 languages.
  - **Email support** — You can now reach the Podhoc team by sending an email directly to the podhoc.com domain.                        
                                                                                                                                        
  ## Improvements                                                                                                                       
                                                                                                                                        
  - **Smoother language switching** — Changing the website language no longer asks you to log in again when you already have an active  
  session.                                                           
  - **Language selector on mobile** — The language picker on the website now displays correctly on phones and tablets.                  
  - **Translation fixes** — Corrected several missing or incorrect labels in the Catalan, English, and Spanish versions of the app.
